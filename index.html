Rail Simulation

At UIC's Electronic Visualization Lab I built the software on the 'RailSim' project. It involved the 3d visualization of a railway simulation in a virtual reality environment known as the CAVE2™. The CAVE2™ is a hybrid system that combines the benefits of both scalable-resolution display walls and virtual-reality systems to create a seamless 2D/3D environment that allows a user to get a highly immersive experience allowing for a detailed analysis of visualized scientific data.
